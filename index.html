Html

HTML (англ. HyperText Markup Language — мова розмітки гіпертексту) — стандартизована мова розмітки документів для перегляду вебсторінок у браузері. Ми використовуємо html для створення каркасу вебсторінок. За допомогою нього ми додаємо на сторінку базові елементи каркасу.

Css

CSS (англ. Cascading Style Sheets, укр. Каскадні таблиці стилів) — це спеціальна мова стилю сторінок, що використовується для опису їхнього зовнішнього вигляду. Ми використовуємо Css для оформлення вебсторінок. За допомогою нього ми задаємо стилі для елементів каркасу.
